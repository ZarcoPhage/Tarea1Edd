REQUERIMENTOS DE USO Y COMPILACIÓN:
--------------------------------------------------------------------

--> COMPILADOR G++/GCC VERSION 10.2.1 O SUPERIOR 
--> PREFERIBLEMENTE OCUPAR LINUX O SISTEMA SIMILAR COMPATIBLE CON G++
--> INCLUIR LOS ARCHIVOS DEL CASO A ANALIZAR EN LA CARPETA "casoT1" INCLUIDA EN LA CARPETA DEL PROGRAMA
    ES DECIR, INCLUIR EL ARCHIVO DE TEXTO .TXT DE LOS SERVICIOS CON EL NOMBRE "servicios.txt" Y EL ARCHIVO BINARIO .DAT CON EL NOMBRE
    "tickets.dat", SOBREESCRIBIR LOS ARCHIVOS O LA CARPETA ENTREGADOS, ASEGURARSE QUE CUMPLEN CON EL FORMATO ENUNCIADO

    ej:

    Tarea1Edd
    - casoT1 
    - - servicios.txt
    - - tickets.dat
    - main.cpp

--------------------------------------------------------------------
COMPILACIÓN Y EJECUCIÓN:
--------------------------------------------------------------------
-> USAR EL SIGUIENTE COMANDO EN TERMINAL EN EL DIRECTORIO DEL PROGRAMA PARA COMPILAR EL CÓDIGO:

    $ g++ main.cpp -o test -Wall

-> USAR EL SIGUIENTE COMANDO EN TERMINAL EN EL DIRECTORIO DEL PROGRAMA PARA EJECUTAR EL PROGRAMA:

    $ ./test
